# Estudos em Java ☕

Este repositório reúne exercícios e exemplos desenvolvidos durante meus estudos em **Java**, com foco em **Programação Orientada a Objetos** e no uso do framework **Collections**.

O objetivo é praticar conceitos fundamentais da linguagem por meio de códigos simples e organizados, utilizados tanto para estudo pessoal quanto para atividades acadêmicas.

---

## 📌 Conteúdos abordados

### 🔹 Encapsulamento
- Criação de classes com atributos privados
- Uso de getters e setters
- Controle de acesso aos dados
- Boas práticas de orientação a objetos

### 🔹 Collections Framework
Exercícios práticos utilizando:
- **List**: armazenamento de dados ordenados
- **Set**: elementos únicos, sem duplicação
- **Map**: estrutura de chave e valor
